====================
--初始化动画系统

initAnime()

初始化动画系统

====================
--添加动画容器

addAnimeContainer(string)

创建一个动画容器

====================
--获得动画容器

getAnimeContainer(string)

获得指定动画容器的闭包
	--addAnime(string,string,number,string,number)
	添加动画对象，并返回一个闭包
	[注释：参数：动画名称，文件夹路径，文件数目，文件格式，动画层]

		--setAxis(number,number,number)
		设置动画的绘制坐标

====================
--获得动画当前帧位置

getAnimePos(string,string)

获得动画容器中的指定对象的帧位置，返回number

====================
--辅助函数，将容器对象排序

sort(table)

将动画容器中的动画对象按照动画层升序排序并保存在一个表中，返回table

====================
--辅助函数，将表转换成容器

toContainer(table)

将表转换成动画容器，返回table

====================
--绘制动画

drawAnime(string,string)

绘制一个指定动画对象，指定为动画时为操作单一动画对象，指定为容器时操作整个容器内的动画对象

====================
--开始动画

staAnime(string,string)

开始播放一个指定动画，指定为动画时为操作单一动画对象，指定为容器时操作整个容器内的动画对象
[注释：动画暂停后再开始可以恢复播放状态，动画结束后再开始将复位动画]

====================
-暂停动画

stopAnime(string,string)

暂停播放一个指定动画，指定为动画时为操作单一动画对象，指定为容器时操作整个容器内的动画对象

====================
--结束动画

endAnime(string,string)

结束播放一个指定动画,指定为动画时为操作单一动画对象，指定为容器时操作整个容器内的动画对象

====================
--破坏动画

desAnime(string,string)

释放一个指定动画或指定容器在系统中的引用，交给GC处理

====================
--设置动画数据

setAnimeData(string)

返回一个指定动画的闭包，用于操作帧数据
	--addAnime(number,table)
	添加一组动画帧
	--desAnime(number)
	破坏指定动画帧
	--altAnime(number,number)
	替换指定动画帧
	--moveAnime(number,number)
	移动指定动画帧

====================
--指定动画显示状态

setAnimeShow(string,string,boolean)

设置一个指定动画的显示状态,指定为动画时为操作单一动画对象，指定为容器时操作整个容器内的动画对象

====================
--设置动画速度

setAnimeSpeed(string,string,number)

设置一个指定动画的显示状态,指定为动画时为操作单一动画对象，指定为容器时操作整个容器内的动画对象

====================
--设置动画播放顺序

setAnimeOrder(string,string,number)

设置一个指定动画的显示状态,指定为动画时为操作单一动画对象，指定为容器时操作整个容器内的动画对象

====================
--设置动画坐标

setAnimeAxis(string,string,table)

设置一个指定动画的绘制坐标,指定为动画时为操作单一动画对象，指定为容器时操作整个容器内的动画对象

====================
--设置动画层

setAnimeLayer(string,string)

返回一个动画操作闭包，用于操作
	--setToTop()
	设置到最高层
	--setToBottom()
	设置到最底层
	--setLayer(number)
	设置当前动画层

====================
--获得动画对象

getAnime(string,string)

返回一个动画操作闭包，用于操作
	--getW()
	获得图像宽
	--getH()
	获得图像高
	--getAxis()
	获得坐标值，返回table
	--getPos()
	获得当前帧位置
	--getNum()
	获得动画帧总数量
	--setAxis(table)
	设置坐标
	--setSpeed(number)
	设置切换动画帧的速度
	--setNowPos(number)
	设置当前播放帧位置
	--setStaPos(number)
	--设置开始播放帧位置
	--setEndPos(number)
	设置结束播放帧位置
	--setAnimeData()
	设置动画的数据
	--staAnime()
	开始播放
	--stopAnime()
	暂停播放
	--endAnime()
	结束播放

====================
--复制动画对象

copyAnime(string,string,string,string)

复制指定动画对象到另一个对象里，指定为动画时为操作单一动画对象，指定为容器时操作整个容器内的动画对象
[注释：参数：被复制的容器，被复制的对象，目标容器，目标对象]


额外注释：
====================
--指定对象

指定对象的区分方式为，传入函数的参数值数目，通常为1时操作的对象为容器，为2时操作的对象为动画对象；
具体情况参照函数具体实现
